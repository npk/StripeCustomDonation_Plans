Steps for Setup

  1. Setup Stripe and receive your custom keys
  2. Insert publishable test key where setPublishableKey is set on script.js page
  3. Insert publishable and secret keys in the config.php file
  4. In your js/script.js - make sure that http://stripe.dev in $('#payment-form').attr('action', 'http://stripe.dev/plans/'+amount+'charge.php'); is pointing to your correct domain
